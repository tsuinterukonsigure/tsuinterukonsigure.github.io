概念引入

BFS（Breadth First Search），即广度优先算法。这个是对于图的一种算法，可以用来求最短路径。对于一个图$G=(V,E)$和一个源顶点$s$，从$s$开始，按照$G$的宽度来盲目搜索节点，若发现目标则终止，它并不考虑结果的可能位置，而是彻底地搜索整张图。
